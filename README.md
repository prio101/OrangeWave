OrangeWave

## A Ghost Theme Named Orange Wave
Version: 1.0.0
###Author: Mahabub Islam
This is a minimal ghost theme with scrolling effect.

## Features
 
------------------
  This is a scrolling effect based ghost theme named Orange Wave
  
  Used:
  * Bootstrap,
  * html5 boilerplate
  * Handlebars
  * Css3
  * Javascript
  * Node Js
